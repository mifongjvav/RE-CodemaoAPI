# 工作室信息

> 获取工作室信息  
> `/web/shops/<workshop_id>` `GET`

## 返回内容

|          key          |            value             |  type  |
| :-------------------: | :--------------------------: | :----: |
|          id           |          工作室 ID           | Number |
|        shop_id        |         工作室识别码         | Number |
|       label_id        |            `未知`            | Number |
|         name          |          工作室名称          | String |
|       logo_url        |            `未知`            | String |
|        n_users        |            `未知`            | Number |
|     manual_score      |            `未知`            | Number |
|      total_score      |          工作室总分          | Number |
|      preview_url      |        工作室图片 URL        | String |
| recommend_preview_url |            `未知`            | String |
|      description      |          工作室介绍          | String |
|        n_works        |        工作室作品数量        | Number |
|     n_contributes     |            `未知`            | Number |
|        n_views        |       工作室被浏览次数       | Number |
|        n_likes        |            `未知`            | Number |
|         level         |          工作室等级          | Number |
|        status         |          工作室状态          | String |
|        ordinal        |            `未知`            | Number |
|   latest_joined_at    |     最后有人加入的时间戳     | Number |
|      created_at       |       工作室创建时间戳       | Number |
|      updated_at       | 工作室主页最后被访问的时间戳 | Number |
