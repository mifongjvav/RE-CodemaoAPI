# 个人信息

> 获取个人信息  
> `/api/user/info` `GET` `需要cookies`

## 返回内容

|     key     |             value              |  type  |
| :---------: | :----------------------------: | :----: |
|     age     |              年龄              |  Number   |
|   avatar    |            头像链接            | String |
| discription |              简介              | String |
|    doing    |           正在做的事           | String |
|    email    |              邮箱              | String |
|    gold     |              金币              |  Number   |
|     id      |            用户 id             |  Number   |
|     sex     |      性别（m 为男，s 为女）      | String |
|    level    | 等级（1 为潜力新星，以此类推） |  Number   |
|     qq      |             qq 号              | String |
